# 龙型宠物美术设计

## 目标

为桌面虚拟宠物建立第一套可量产的龙型 PNG 素材。角色需从幼体的 QQ 宠物式可爱感，逐步成长为数码兽式的帅气与力量感，同时保持同一角色的清晰辨识度。

## 角色锚点

- 主色：橙红色；腹甲：奶白色；眼睛：琥珀色。
- 头部：短吻部、两根向后弯的暖褐色角与一根较小的中央角；四足、可见短尾。
- 阶段间不改变角色种族、配色或角的基本形状，只扩展体型、翅膀、鳞甲和纹样。

## 五阶段

| 阶段 | 视觉重点 |
| --- | --- |
| baby | 圆胖比例、大眼、短肢、角芽和小折翼；适合桌面互动。 |
| adult | 四足站立；身体稍修长，翅膀、鳞片与角开始成型。 |
| perfect | 铠甲鳞片与王冠感双角更明确，体格强壮但轮廓紧凑。 |
| ultimate | 具有龙皇气质；翼膜、护甲和神圣纹样更精致，仍保持完整全身轮廓。 |
| superUltimate | 神性终态；增加克制的发光纹样与翼膜细节，不使用环境、烟雾或额外器官。 |

## 生成与输出约束

- 每张为 1:1、1024px、全身居中，四周留出至少 12% 安全边距。
- 成体至终极体固定为 3/4 正面、略向右的站姿；幼体采用参考图的正面微 3/4 坐姿；不使用飞行、盘绕或夸张透视。
- Midjourney 先生成平整洋红色（#FF00FF）背景图；之后去背并导出带 alpha 的 PNG。
- 禁止背景、地面、投影、文字、水印、卡牌边框、武器、环境粒子和独立漂浮特效。
- 身体 PNG 不携带元素效果；火、风、地、水、光效果由独立 elements 图层提供。

## 接入路径

五张输出分别保存为：

`public/assets/pets/bodies/dragon/{baby,adult,perfect,ultimate,superUltimate}.png`

每张必须能与同阶段的元素 PNG 在 `PetDisplay` 中以 `object-fit: contain` 同尺寸叠加。

## 验收

- 五张图一眼能认出是同一只龙的进化链。
- 缩放至约 80px 时，眼睛、双角、四足轮廓和蓝白配色仍清楚。
- 图像四角透明，边缘无洋红溢色。
- 页面在存在 body 与 element 两张对应图片时加载 PNG 合成模式；任何一张缺失则继续降级为 SVG。
